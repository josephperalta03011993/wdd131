W05 Project: Website Planning Document

1. Site Name

Biking Adventures

This name represents a site focused on biking adventures, routes and tips.

Optional Domain Available: biking-adventure.com

2. Site Purpose

The website will showcase biking routes, personal ride experiences, bike maintenance tips, and gear recommendations. It will also features a gallery of biking photos and videos.

3. Scenarios

What is a good gear when riding uphill?

How can I better maintain my mountain bike?

4. Color Schema

#FF7043 (Sunset Orange) – Primary action color
#212121 (Charcoal Black) – Text and accents
#FBC02D (Adventure Yellow) – Highlights & excitement
#FAFAFA (Soft White) – Clean background

5. Typography

Roboto - for headings

Sans-serif - for text and paragraphs
